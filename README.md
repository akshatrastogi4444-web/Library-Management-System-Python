# Enhanced Library Management System
A Python-based system to manage books, track issues/returns, and calculate late fines.

## How to Run
1. Download `main.py`.
2. Run `python main.py` in your terminal.

## Features
* **Automated Fine Calculation**: Charges Rs. 5 per day for late returns.
* **Columnar Display**: Clean table view for all book records.
* **Search**: Find books by ID, Title, or Author.
